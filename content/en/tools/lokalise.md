# Lokalise

Lokalise is a cloud-based translation and localization management platform designed specifically for teams that deliver software, apps, and websites in multiple languages. With automation tools and AI-powered features, Lokalise enables efficient collaboration between developers, translators, and project managers. The platform supports numerous file formats and integrations to simplify and accelerate the localization process.

## Who is Lokalise suitable for?

Lokalise is aimed primarily at companies and teams that develop or run multilingual software products. This includes:

- Software developers and product teams that want to internationalize their apps and websites.
- Translation agencies and freelancers who want to manage translation projects efficiently.
- Project managers who want to keep track of localization processes.
- Companies of all sizes, from startups to large enterprises, looking for a central platform for language management.

Lokalise is especially beneficial for teams that value automation, integration into existing workflows, and a high degree of flexibility in collaboration.

## Key Features

- **Cloud-based translation platform:** Access from anywhere, no local installation required.
- **Automated workflows:** Task assignment, reminders, and status tracking.
- **AI-powered translation assistance:** Suggestions for machine translation and terminology management.
- **Support for numerous file formats:** JSON, XML, CSV, YAML, Android/iOS strings, and more.
- **Integrations:** GitHub, GitLab, Slack, Jira, Figma, and other tools for seamless integration.
- **In-app editor:** Real-time collaboration and translation previews.
- **Glossary and terminology management:** Consistent use of technical terms.
- **API and CLI tools:** For custom automation and integration into CI/CD pipelines.
- **Version control:** Change tracking and easy rollback.
- **User and permission management:** Fine-grained access control for teams.

Yes, the platform supports AI-powered translation suggestions and can be connected to various machine translation services.
