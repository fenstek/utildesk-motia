# Avigilon

Avigilon is a video security platform for organisations that need to monitor sites, investigate incidents and use analytics to identify relevant events faster. It combines cameras, management software and AI-assisted video analysis.

The platform is best evaluated as part of a broader physical security setup rather than as a standalone video tool.
