# RightInbox

RightInbox is best understood less as a feature list and more as a specific workflow: email follow-ups and send scheduling directly in Gmail. Its practical value shows up where you need to manage replies, reminders, and later send times from the inbox without pushing every decision into separate tools.

What matters most in the evaluation is the question of which follow-ups are truly valuable and which only create noise. If that point remains unresolved, even a strong tool can quickly seem bigger than its real benefit.

## Who is RightInbox for?

RightInbox is suitable for small teams, sales, recruiting, and personal follow-up routines. If you only need a quick one-off action, keep the effort small and first check which follow-ups are truly valuable and which only create noise.

The tool is less suitable when this warning sign becomes visible during the pilot: too many reminders that merely fill the inbox in a different way. In such cases, a leaner process is often more sensible than a big platform decision.

- **Automatic follow-ups:** reminders for unanswered emails are set up automatically so important messages are not forgotten.

- **Email tracking:** notifications when a sent email has been opened or clicked.
- **Notes:** the option to add internal notes to emails that are visible only to the user.

## FAQ

**1. Does RightInbox only work with Gmail?**
Yes, RightInbox is specifically designed for Gmail and works as a browser extension that is integrated directly into the Gmail inbox.

**2. Is there a free version of RightInbox?**
There is a free basic plan with limited functions, ideal for testing. A paid subscription is required for advanced features.

**3. Can I automatically send emails at specific times with RightInbox?**
Yes, one of the core features is email scheduling, which lets you send messages automatically at a desired time.

**4. Is my data processed securely when using RightInbox?**
RightInbox places value on privacy and security, but users should still review the relevant privacy policies, especially when handling sensitive company data.

**5. Is RightInbox also suitable for teams?**
Yes, there are team plans that offer features for shared templates, tracking, and collaboration.

**6. Which browsers are supported?**
RightInbox is compatible with common browsers such as Google Chrome, Firefox, and Microsoft Edge.

**7. Can I create and save email templates with RightInbox?**
Yes, the tool allows you to create, save, and reuse templates, which makes it faster to send frequently used emails.

**8. Is there a mobile app for RightInbox?**
RightInbox is currently mainly available as a browser extension; there is no standalone mobile app.

**9. How should RightInbox be tested?**
Best with a small, real scenario from your own daily work. Check whether the tool helps manage replies, reminders, and later send times in the inbox, and whether the results are usable without much rework.

**10. What is the most common stumbling block with RightInbox?**
The most common stumbling block is starting too broadly. Before rollout, it should be clear which follow-ups are truly valuable and which only create noise; otherwise, the benefit is hard to evaluate.
