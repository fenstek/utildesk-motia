# H5P

H5P is an open-source framework for creating interactive learning content. It is used to build quizzes, interactive videos, course presentations, timelines, drag-and-drop exercises, flashcards, and other reusable learning modules.

## Who is H5P for?

H5P is useful for educators, instructional designers, universities, schools, training teams, and developers who need interactive educational content that can be embedded into learning platforms and websites.

## Key features

- Many ready-made interactive content types.
- Browser-based authoring workflow.
- Embedding into learning platforms and websites.
- Open-source foundation and broad education community.
- Useful for quizzes, videos, exercises, presentations, and self-checks.
- Integration with platforms such as Moodle, WordPress, and Drupal.

## What really matters in daily use

In day-to-day work, H5P is less about having every edge feature and more about whether the team understands where work starts, who reviews it, and how results move forward. A useful setup defines roles, naming rules, and the most important handover points before adoption.

H5P is strongest when it reduces friction in an existing workflow instead of creating a second place to maintain. Before rolling it out widely, test it with real examples: which task becomes faster, which decision becomes clearer, and which manual check should intentionally remain?

## Workflow Fit

H5P fits best into a workflow with a clear input, a traceable work step, and a defined finish line. Small teams can usually keep the process lightweight; larger organizations should also define permissions, approvals, and integrations.

If H5P becomes just another account without ownership, the value fades quickly. Give it a clear place in the existing stack: what enters the tool, what gets decided there, and where the result goes next.

## Privacy & Data

Before adopting H5P, clarify which data will enter the tool and whether source code, logs, customer data, and technical metadata are involved. The more sensitive the material, the more important permissions, retention rules, export options, and a documented decision on what should stay outside the tool become.

For European teams evaluating H5P, data processing agreements, hosting information, and deletion processes are also worth checking. This is not a substitute for legal advice, but it avoids the common mistake of introducing H5P before the data path is understood.
