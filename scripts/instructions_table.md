# Инструкции по добавлению инструментов в таблицу

## 1. Скрипт для добавления инструментов в таблицу
Скрипт: `scripts/sheet_ai_autogen_9_strict_v2.mjs`
- Основная цель: добавление инструментов в таблицу Google Sheet.
- Логика работы:
  1. Генерирует список инструментов с использованием OpenAI (только AI-first продукты).
  2. Производит валидацию через Wikidata.
  3. Проверяет уникальность по различным признакам: topic, slug, wikidata_id, official_url.
  4. Добавляет инструменты в таблицу, если они не дублируются.

## 2. Скрипт для добавления популярных инструментов вручную
Скрипт: `scripts/sheet_add_known_popular.mjs`
- Этот скрипт добавляет вручную несколько популярных инструментов, таких как:
  - Claude
  - Pika
  - Notion AI
  - Copy.ai
- Он проверяет, есть ли уже данные в таблице, и если инструмент отсутствует — добавляет его.
- Для каждого инструмента задаётся категория через `categoryFallback()`.

## 3. Использование официальных фавиконов
- Для каждого инструмента, в таблице указывается URL официального фавикона (например, через сервисы типа [favicon.io](https://favicon.io/)).
- Это позволяет избежать генерации изображений и использовать уже существующие логотипы.

## 4. Регулярный запуск с cron
В будущем можно настроить cron, который будет запускать скрипт для добавления популярных инструментов и валидировать их через Wikidata.
- Периодичность запуска: например, раз в день.
- Статус работы будет выводиться в логи, а новые инструменты добавляться в таблицу.
