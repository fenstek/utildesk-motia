# Инструкции по работе пайплайна

## 1. Как работает пайплайн автоматической генерации контента
Пайплайн состоит из нескольких этапов:

### Этап 1: Добавление инструмента в таблицу
- Скрипт `scripts/sheet_ai_autogen_9_strict_v2.mjs` генерирует список инструментов и проверяет их на уникальность.
- Инструменты добавляются в таблицу Google Sheet на основе их популярности и уникальности.

### Этап 2: Генерация Markdown страниц для инструментов
- После того как инструменты добавлены в таблицу, следующий шаг — генерация Markdown файлов для каждой строки с инструментом.
- Скрипт использует `scripts/generate_tool_md.mjs` для автоматической генерации контента.

### Этап 3: Генерация изображений для инструментов
- Для каждого инструмента генерируются изображения через скрипт `scripts/generate_tool_image.mjs`.
- Изображения оптимизируются в формат `.webp` и добавляются в соответствующие Markdown страницы.

### Этап 4: Заливка контента на сайт
- После генерации Markdown страниц и изображений контент автоматически деплоится на сайт через GitHub Actions и Cloudflare Pages.

## 2. Регулярный запуск пайплайна с cron
Для автоматизации всех процессов можно настроить cron на выполнение пайплайна:
- Каждый день будет запускаться пайплайн, который:
  1. Добавляет новые инструменты в таблицу.
  2. Генерирует новые страницы Markdown.
  3. Генерирует новые изображения.
  4. Деплоит обновлённый сайт.
